# Galaxy Catalog

A Flask-based web application for browsing and managing taxonomic catalog data with an interactive interface.

## Features

- **Taxonomy Browser**: Hierarchical tree navigation for browsing taxonomy collections
- **Interactive Data Table**: Server-side paginated table view powered by Grid.js
- **Gallery View**: Card-based layout for visual browsing
- **Advanced Search**: Full-text search across species data
- **Collection Filtering**: Filter items by taxonomy collection
- **Responsive Design**: Mobile-friendly interface with collapsible sidebar
- **RESTful API**: JSON API endpoints for data access
